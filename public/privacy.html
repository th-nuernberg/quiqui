Datenschutzerklärung

Verantwortlicher

Verantwortlicher für die Datenverarbeitung auf dieser Website ist die im Impressum genannte Person bzw. Einrichtung.

Zweck der Anwendung

Diese Website stellt eine webbasierte Lehr- und Lernanwendung zur Verfügung. Die Nutzung ist ohne Benutzerkonto möglich.

Es werden keine Namen, E-Mail-Adressen oder sonstigen personenbezogenen Profildaten erhoben.

Verarbeitung innerhalb der Anwendung

Von Teilnehmenden eingegebene Antworten werden ausschließlich zur Durchführung der jeweiligen Sitzung verarbeitet.

Die Daten werden nur temporär im Arbeitsspeicher des Servers vorgehalten und nach Beendigung bzw. Ablauf der Sitzung automatisch gelöscht. Eine dauerhafte Speicherung erfolgt nicht.

Zur technischen Zuordnung aktiver Verbindungen werden temporäre Sitzungskennungen verwendet. Diese dienen ausschließlich dem Betrieb der Anwendung und werden nicht dauerhaft gespeichert.

Lehrkräfte können öffentliche GitHub-Repository-URLs angeben, um Inhalte für die Anwendung bereitzustellen. Diese URLs werden ausschließlich zur Bereitstellung der Funktionalität verarbeitet.

Im Browser der Lehrkraft wird eine zufällig erzeugte technische Kennung im Local Storage abgelegt. Sie dient ausschließlich dazu, das eigene Gerät bei erneutem Aufruf wiederzuerkennen und so zu verhindern, dass zwei Lehrkräfte versehentlich dieselbe laufende Sitzung überschreiben. Diese Kennung enthält keine personenbezogenen Daten, wird nicht an Dritte übermittelt und verbleibt ausschließlich lokal im Browser, bis sie dort gelöscht wird.

Hosting und Server-Logfiles

Beim Aufruf dieser Website verarbeitet der Hosting-Anbieter technisch erforderliche Zugriffsdaten in Server-Logfiles. Hierzu können insbesondere die IP-Adresse, Datum und Uhrzeit des Zugriffs, Informationen zum verwendeten Browser sowie die aufgerufenen Seiten gehören.

Die Verarbeitung erfolgt zum sicheren und zuverlässigen Betrieb der Website auf Grundlage von Art. 6 Abs. 1 lit. f DSGVO.

Cookies und lokaler Speicher

Diese Website verwendet keine Cookies. Im Browser der Lehrkraft wird lediglich eine zufällige technische Kennung im Local Storage abgelegt (siehe oben, „Verarbeitung innerhalb der Anwendung“); diese ist für den Betrieb der Anwendung technisch notwendig und erfordert daher keine gesonderte Einwilligung.

Analyse- und Trackingdienste

Es werden keine Analyse-, Tracking- oder Marketingdienste eingesetzt.

Betroffenenrechte

Soweit personenbezogene Daten verarbeitet werden, stehen den betroffenen Personen die gesetzlichen Rechte nach der DSGVO zu.

Privacy Policy

Controller

The controller responsible for data processing on this website is the person or institution named in the Impressum.

Purpose of the Application

This website provides a web-based teaching and learning application. No user account is required to use it.

No names, email addresses, or other personal profile data are collected.

Data Processing Within the Application

Answers submitted by participants are processed solely for the purpose of conducting the respective session.

Data is held only temporarily in the server's memory and is automatically deleted when the session ends or expires. No permanent storage takes place.

Temporary session identifiers are used for the technical assignment of active connections. These serve exclusively to operate the application and are not stored permanently.

Hosts may provide public GitHub repository URLs to supply content for the application. These URLs are processed solely to provide that functionality.

A randomly generated technical identifier is stored in the host's browser using Local Storage. Its sole purpose is to recognise the same device on a later visit, preventing two hosts from accidentally overwriting each other's running session. This identifier contains no personal data, is never transmitted to third parties, and remains entirely on the local device until deleted there.

Hosting and Server Log Files

When this website is accessed, the hosting provider processes technically necessary access data in server log files. This may include the IP address, date and time of access, information about the browser used, and the pages visited.

Processing is carried out for the secure and reliable operation of the website on the basis of Art. 6(1)(f) GDPR.

Cookies and Local Storage

This website does not use cookies. The host's browser stores only a random technical identifier in Local Storage (see "Data Processing Within the Application" above); this is technically necessary for the application to function and therefore does not require separate consent.

Analytics and Tracking

No analytics, tracking, or marketing services are used.

Data Subject Rights

To the extent that personal data is processed, data subjects are entitled to the statutory rights under the GDPR.
